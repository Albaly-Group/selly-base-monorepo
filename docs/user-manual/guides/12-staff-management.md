# Staff Management

Coordinate and manage staff members within your organization for efficient teamwork and data management.

## Overview

Staff Management allows you to:
- View staff members
- Add new staff
- Edit staff details
- Manage staff permissions
- Track staff activity

**Available to:** Staff (view only), Customer Admin, Platform Admin

**Available to:** Customer Admin, Platform Admin

3. Select new role

### Managing Staff Status

**Deactivate Staff:**
- Temporarily disable access
- User can be reactivated
- Data and history preserved

**Reactivate Staff:**
- Restore access
- Same credentials
- Previous settings intact

**Remove Staff:**
- Permanently delete account
- Cannot be undone
- Data transferred to manager

## Staff Permissions

### Permission Levels

**View Only:**
- Browse companies
- View lists
- Run searches
- No modifications

**Standard Staff:**
- All view permissions
- Create lists
- Add/remove companies
- Export data (within limits)

**Senior Staff:**
- All standard permissions
- Import data
- Generate reports
- Manage own team

**Staff Administrator:**
- All staff permissions
- Manage other staff
- Configure policies
- View all activities

### Customizing Permissions

**Grant/Revoke:**
1. Open staff profile
2. Go to "Permissions" tab
3. Toggle permissions
4. Save changes

**Permission Categories:**
- Company Management
- List Operations
- Import/Export
- Reports
- User Management
- Settings

## Troubleshooting

### Cannot Add Staff

**Possible reasons:**
- User limit reached
- Duplicate email
- Insufficient permissions
- Invalid data

**Solutions:**
- Check organization limits
- Verify email unique
- Contact admin
- Correct form errors

### Staff Cannot Login

**Common issues:**
- Account not activated
- Incorrect credentials
- Account locked
- Browser issues

**Solutions:**
- Resend welcome email
- Reset password
- Unlock account
- Check status

### Missing Staff Data

**Why data missing:**
- Filters applied
- Deactivated account
- Different organization
- Sync delay

**Solutions:**
- Clear filters
- Check status
- Verify organization
- Wait for sync

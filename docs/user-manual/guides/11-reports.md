## Troubleshooting

### Report Won't Generate

**Possible Causes:**
- No data for parameters
- Invalid date range
- Server timeout
- Permission issues

**Solutions:**
- Adjust date range
- Broaden filters
- Reduce detail level
- Check permissions

### Missing Data

**Why data might be missing:**
- Filters too restrictive
- Date range incorrect
- No access to data
- Data not yet updated

**Solutions:**
- Review filter settings
- Check date range
- Verify access permissions
- Wait for data sync

### Export Failed

**Common issues:**
- Report too large
- Format not supported
- Browser timeout
- Memory issues

**Solutions:**
- Reduce data size
- Try different format
- Use scheduled export
- Contact administrator

The Dashboard is your home base in Selly Base, providing at-a-glance metrics, recent activity, and quick access to key features.

## Accessing the Dashboard

**Navigation:** Automatic after login, or click "Dashboard" in main menu

**URL:** `/dashboard`

**Available to:** All user roles

## Dashboard Components

### Key Metrics Cards

The dashboard displays key performance indicators (KPIs) relevant to your role:

#### For Platform Admins
- **Active Tenants:** Number of customer organizations (e.g., 12 active)
- **Total Users:** Platform-wide user count (e.g., 1,247 users)
- **Shared Companies:** Companies in shared database (e.g., 45,231)
- **System Health:** Platform uptime percentage (e.g., 99.9%)

#### For Customer Admins
- **Organization Users:** Users in your organization
- **Active Lists:** Number of active company lists
- **This Month's Exports:** Export count for current month
- **Database Size:** Companies in organization database

#### For Staff
- **My Lists:** Number of lists you've created
- **Companies Added:** New companies this month
- **Exports Created:** Your exports this month
- **Search Activity:** Your search count this week

#### For Users
- **My Lists:** Personal lists count
- **Recent Searches:** Number of saved searches
- **This Week's Activity:** Actions performed
- **Quick Access:** Favorite features

### Recent Activity Feed

Shows chronological list of recent actions:

**Activity Types:**
- Companies added to lists

- Exports completed
- Imports processed

**Activity Details:**
- Action description
- User who performed action
- Timestamp

**Filtering:**
- Filter by action type
- Filter by user
- Filter by date range
- Search activity feed

### Charts and Visualizations

#### Activity Chart
- **Type:** Line or bar chart
- **Shows:** Activity over time (last 30 days)
- **Metrics:** Searches, exports, imports, list operations
- **Interactive:** Hover for details, click to drill down

#### Company Distribution Chart
- **Type:** Pie or donut chart
- **Shows:** Companies by industry, location, or size
- **Interactive:** Click segment to filter

#### Lead Score Distribution
- **Type:** Histogram
- **Shows:** Distribution of companies by lead score
- **Use:** Identify high-value prospects

#### User Activity Chart (Admins)
- **Type:** Bar chart
- **Shows:** Most active users
- **Metrics:** Actions per user
- **Period:** Selectable (week/month/quarter)

## Dashboard Refresh and Real-Time Updates

### Auto-Refresh

Dashboard automatically refreshes:
- Metrics: Every 5 minutes
- Activity Feed: Every 30 seconds
- Notifications: Real-time (when supported)
- Charts: Every 5 minutes

### Manual Refresh

Force refresh anytime:
- Click refresh icon (top-right)
- Press F5 or Ctrl+R
- Pull down to refresh (mobile)

## Troubleshooting

### Dashboard Not Loading

**Check:**
- Internet connection
- Browser compatibility
- Clear browser cache
- Try incognito mode
- Check system status page

### Data Not Updating

**Solutions:**
- Click manual refresh
- Check connection indicator
- Verify permissions
- Log out and back in
- Contact support if persistent

### Metrics Seem Incorrect

**Verify:**
- Date range selected
- Filters applied
- Your role permissions
- Data sync status
- Compare with reports page

### Missing Widgets

**Possible Reasons:**
- Widget hidden in preferences
- Insufficient permissions
- Feature not available for your plan
- Customization not saved

**Solution:**
- Check dashboard preferences
- Reset to default layout
- Contact administrator

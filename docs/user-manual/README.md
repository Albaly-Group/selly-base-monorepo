## Welcome to Selly Base

Selly Base is a comprehensive B2B prospecting and lead management platform designed to help businesses search, filter, and manage company databases with advanced features including lead scoring, multi-tenant support, and comprehensive audit capabilities.

This user manual provides step-by-step guidance for all user roles and covers every feature and function of the platform.

## User Roles Overview

### Platform Admin
**Who:** Albaly platform administrators  
**Access:** Complete platform access across all organizations  
**Primary Functions:** Tenant management, shared data, platform analytics

### Customer Admin
**Who:** Organization administrators  
**Access:** Full access within their organization  
**Primary Functions:** User management, policies, organization settings

### Staff
**Who:** Database managers and analysts  
**Access:** Data management and reporting  
**Primary Functions:** Company search, lists, imports, exports, reports

### User
**Who:** Regular end users  
**Access:** Read-only with limited exports  
**Primary Functions:** Company search, personal lists, basic analytics
